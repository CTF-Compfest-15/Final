# Final CTF CompFest 2023

Problemset for Final CompFest 2023

## Challenge Folder Structure
```
problem-slug/
 ├─ attachment/
 │   └─ archived_attachment (publish to participant)
 ├─ test/
 │   ├─ (file needed for testing)
 │   └─ test.py
 ├─ src/
 ├─ writeup/
 │   ├─ (some script)
 │   └─ README.md (vuln list, attack payload, and patching)
 ├─ archived_attachment (publish to participant)
 ├─ challenge.yml
 ├─ docker-compose.yml
 └─ patchrule.yml
```
